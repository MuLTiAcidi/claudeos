# ClaudeOS Desktop Edition

Full Linux desktop with ClaudeOS system tray app and desktop integration.

## Requirements
- Ubuntu 22.04+ Desktop
- 2GB RAM minimum
- Display/monitor connected

## Install
```bash
# Install on existing Ubuntu Desktop
curl -fsSL https://raw.githubusercontent.com/herolind/claudeos/main/install.sh | sudo bash
sudo bash /opt/claudeos/editions/desktop/install.sh
```

## Features
- Everything from Server + Web Dashboard editions
- Desktop notification popups for alerts
- System tray icon showing health status
- ClaudeOS app in application menu
- Auto-starts dashboard on login
- Desktop widget showing CPU/RAM/Disk
